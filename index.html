Presentation Content

This presentation goes over the shiny project delivered in this class. The project uses mtcars data and allows an interactive selection of predictors from the dataset to relate with the MPG.

  • Illustration of the input selection to choose the MPG predictors
  • The input is processed at the server to validate selections exist
  • The output chart is displayed

Input parameters

The project uses group select input to allow the user to select the predictor(s) that needs to be researched.

Predictors selection validation

The user clicks on submit button at which point the input parameters get sent to the server, the server validates the input and displays a message if no selections were made.

The output chart

If entry is valid the chart gets displayed using googleVis. The chart shows a trend exponential fit for the first selected parameter.
